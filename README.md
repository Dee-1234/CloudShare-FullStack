# The Vision
CloudShare is designed to simplify personal file management. It provides a seamless, "Google Drive-style" experience where users can organize, protect, and retrieve their digital assets through a clean, intuitive web interface.

# Core Functionality
1. Dynamic Dashboard: A single-page interface that switches between "My Drive," "Starred," "Recent," and "Trash" without page reloads.
2. Intuitive UX: Implementation of HTML5 Drag & Drop API for instant uploads and real-time state updates for file starring.
3. Smart Filtering: Advanced logic that automatically highlights the last two uploaded files in the "Recent" tab and segregates "Starred" items.
4. Two-Stage Deletion: Features a "Soft Trash" system, allowing users to move files to a recovery area before permanent removal from the server.
5. Secure Architecture: Backend protection using JWT (JSON Web Tokens) to ensure file privacy and secure data transmission.

# Technical Highlights
1. Frontend: React (Hooks, Context, Axios), CSS-in-JS for modular styling.
2. Backend: Java, Spring Boot, Spring Security (JWT), Maven.
3. Storage: Integrated with Spring's FileSystemResource for efficient binary data handling.
